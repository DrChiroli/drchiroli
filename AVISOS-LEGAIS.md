# Avisos importantes

Este projeto deve ser tratado como conteúdo educativo, estético e de autocuidado.

Não apresente o conteúdo como prescrição médica, dermatológica, nutricional, hormonal, psicológica ou farmacológica.

Evite orientar uso de medicamentos, hormônios, peptídeos, anabolizantes, isotretinoína, finasterida, minoxidil oral ou qualquer tratamento individualizado sem avaliação profissional.

Frases seguras:

- "Procure um dermatologista se houver acne persistente."
- "Procure um médico antes de usar qualquer medicamento."
- "Resultados variam de pessoa para pessoa."
- "O conteúdo tem finalidade educativa."
- "Minha experiência pessoal não substitui avaliação profissional."

Frases perigosas:

- "Faça isso que vai resolver sua acne."
- "Esse remédio funciona para todos."
- "Resultado garantido."
- "Você vai ficar bonito em X dias."
- "Não precisa procurar médico."
